# panglyph/

Universal Unicode 17 font releases from `fontist/panglyph`.

## Contents

- `manifest.json` — latest version + version index
- `v<X.Y.Z>/` — per-release directory containing:
  - `panglyph-unicode<N>.ttf`
  - `panglyph-unicode<N>.woff2`
  - `panglyph-unicode<N>.otf`
  - `coverage-report.json`
  - `source-manifest.json` (OFL provenance per source font)

## Sync

Triggered by `fontist/panglyph` tag push (`v*` tags).
