= fontist-archive-public

Public archive of Unicode coverage data, open-license WOFF specimens,
ucode Unicode data, and panglyph universal-font releases for the Fontist project.

== Purpose

This is the *public consumer* of multiple data streams:

* ALL coverage data (metadata is public — even for proprietary fonts)
* Open-license WOFF specimens ONLY (OFL, Apache, MIT, etc.)
* ucode Unicode data (per-block codepoint feeds + universal glyph set)
* panglyph releases (universal Unicode 17 font)

It does NOT contain:

* Raw font files (proprietary — in private repo only)
* WOFF specimens for macOS/Microsoft fonts (proprietary licenses)
* Build code (this is a consumer, not a builder)

== Structure

----
coverage/{formula_slug}/{PSName}.yaml   ucode audit output (ALL fonts)
  google/abeezee/ABeeZee-Regular.yaml      one YAML per font face
  manual/inter/Inter-Bold.yaml             one per TTC face
  macos/font8/akaya/AkayaKannada.yaml      (coverage IS public even for macOS)

woff/{formula_slug}/{PSName}.woff2     WOFF specimens (open-license ONLY)
  google/abeezee/ABeeZee-Regular.woff2     named from formula post_script_name
  (NO macos/ directory — proprietary)

fonts.json                              Font registry (canonical name → formula slugs)
font-metadata.json                      Per-face metadata (weight, style, etc.)

unicode/                                ucode Unicode-only artifacts
  block-feed/                           compact per-block feed (consumed by fontist.org)
    unicode-blocks.json
    unicode-version.json
    unicode/blocks/<slug>.json
  universal-glyph-set/                  one SVG per codepoint
    manifest.json                       version, counts, generatedAt
    entries/U+XXXX.json                 per-glyph provenance
    glyphs/U+XXXX.svg
  codepoints-{version}.tar.zst          per-codepoint detailed JSONs (release asset)
  codepoints-index.json                 quick lookup index

panglyph/                               panglyph universal-font releases
  manifest.json                         latest version + version index
  v17.0.0/                              per-release directory
    panglyph-unicode17.ttf
    panglyph-unicode17.woff2
    panglyph-unicode17.otf
    coverage-report.json
    source-manifest.json                OFL provenance per source font
----

== Sync pipeline

Three independent pipelines feed this repo:

[cols="1,2,2", options="header"]
|===
|Source |What it syncs |Trigger

|fontist-archive-private |coverage/ + woff/ + fonts.json |per-formula CI runs
|fontist/ucode |unicode/block-feed + unicode/universal-glyph-set |per-Unicode-version publish workflow
|fontist/panglyph |panglyph/v{X.Y.Z}/ + manifest.json |per-release tag
|===

Each pipeline syncs independently to a disjoint directory, so they can
run concurrently without git conflicts.

== Data separation

[cols="1,1,1,2", options="header"]
|===
|Data |Private |Public |Reason

|Raw font files |✅ ALL |❌ NEVER |Contains proprietary fonts
|Coverage YAML |✅ ALL |✅ ALL |Metadata only, no font data
|WOFF specimens |✅ ALL |✅ Open-only |WOFF is a usable font file
|ucode Unicode data |(n/a — public source) |✅ ALL |Pure Unicode data, no fonts
|panglyph universal font |(n/a — assembled from OFL sources) |✅ ALL |OFL
|Build code |✅ |❌ |Public is consumer only
|===
